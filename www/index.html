Requirements

If you are reading this HTML file, it means that you've already set it all up client-side.

On the server-side, you require:

  • Node.js

    And the following packages:

    • express

      expressjs.com

      npm install express
    • sockets.io

      socket.io

      npm install socket.io
    • mysql

      The official MySQL driver provided by Oracle.

      npm install mysql

We are also additionally using the following frameworks and technologies (which are already installed or is not managed by a package manager of any sort).

  • FontAwesome
    fontawesome.com
  • jQuery
    jquery.com
  • Materialize (Material UI Framework)
    materializecss.com

Getting Started

Simply start the Node.js server with the root folder as the context. You are currently at /www/index.html

node app.js

And finally, click on the " Start App" button, or go to localhost:3000/app

Oh yeah...

When you compile the Angular-CLI app, compile it with a different root path, otherwise it'll look for a /main.js that does not exist in /www/main.js.
To do so, simply compile it with the following arguments:

ng build --base-href /app/

Or for production:

ng build --prod --base-href /app/

Source: https://stackoverflow.com/questions/45229785/absolute-path-in-angular-cli

Or, if you'd rather develop and have it pushed to the live server...

ng build --base-href /app/ --watch

And it'll ng serve to the dist folder.n
